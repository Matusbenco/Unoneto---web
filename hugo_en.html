HUGO

Modern, fast, and targeted information for citizens of 21st-century cities and municipalities.

What is HUGO and how does it work?

HUGO is a revolutionary information system that replaces outdated municipal broadcasting. Instead of unreliable and hard-to-hear loudspeakers, it allows municipalities to send important announcements directly to citizens' mobile phones. The system works as a mobile application but can also deliver messages via SMS or email, reaching all age groups.

Main Advantages of the HUGO System

Using artificial intelligence, HUGO can automatically convert written text into a professional-sounding voice announcement. The mayor or an authorized employee can create and send an announcement from a computer or mobile phone in just a few seconds.
